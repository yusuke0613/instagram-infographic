**鉄則**: 冒頭3秒で「自分ごと化」させるには、数値インパクト＋逆説/ネガ事実の組合せが強い。

**Why:**
- 内製SE/情シス層は「業界水準を知りたい」志向が強く、数値ファクト（通過率37.3%、求人倍率10.4倍、SaaS平均207サービス等）に強く反応する
- 「経験5年でも落ちる」「6割は落ちる」など、現状肯定を一度ひっくり返す書き方が保存・コメント率を押し上げる
- 一方、汎用的な「○○のススメ」「○つの方法」だけでは弱い

**How to apply:**
- Reels台本/Threads/Instagram図解の3点全てで、冒頭にネガ事実 or 衝撃数値を1行で置く
- 数値は1つに絞る（複数並べるとスクロールされる）
- 逆説の後に「理由はシンプルで…」と短く回収すると離脱率が下がる
- ハイプ語（「神」「最強」「絶対」）は内製SE/情シス層には逆に刺さらないので避ける
