社内SEコンテンツの数値リサーチで毎回使う一次情報源と、その得意領域。

## 政府・公的機関
- **経産省「IT人材需給に関する調査」** — 2030年79万人不足の出典。マクロ供給ギャップ／先端IT人材 vs 従来型IT人材の二分。
- **総務省 情報通信白書** — 業種別 生成AI活用率／前年比。年1回更新（春〜夏）。
- **IPA「DX動向」シリーズ** — DX人材の量・質、内製化比率（企画65% / 開発17% / テスト23% など）、日米独比較。
- **厚労省 有効求人倍率（一般職業紹介）** — 「情報処理・通信技術者」の正式な公的求人倍率。日経が引用してくる元ネタ。

## 民間調査（信頼度高）
- **NRI「IT活用実態調査」** — CIO/役職者対象。生成AI導入率の年次推移（2023→24→25：33.8→44.8→**57.7%**）。最も意思決定密度が高い。
- **JUAS「企業IT動向調査」** — 言語系生成AI導入率41.2%（2025）。CIO/情シス対象、伝統的に IT 投資動向の定点。
- **PwC「生成AIに関する実態調査」** — 5カ国比較・効果実感率（日本10% vs 米国45%）。日本の運用フェーズの詰まり方を語る時の鉄板ソース。
- **INTAGE BtoB調査** — ビジネスパーソン個人レベルの生成AI利用率。業種別×職位別の二極化を見るならここ。
- **ICT総研** — 生成AI法人利用社数の年次予測（2024：32.7万社 → 2026：50.1万社）。市場規模を語る時。

## 転職市場
- **レバテック プレスリリース／調査** — IT特化の高倍率（求人倍率10.4倍、PM31倍、セキュリティ42.6倍、生成AI関連410%増）。インパクト数値が欲しい時。
- **doda「ITエンジニア中途採用マーケットレポート」** — 全体俯瞰の求人倍率（2.70倍程度）、社内SE登録者比率・入社決定職種比率。
- **マイナビキャリアリサーチLab** — 中途採用総括レポート。年次トレンド全体把握に。
- **社内SEナビ メディア** — 社内SE特化の年収レンジ（中央値）。エージェント発の推計込みなので「中信頼」扱いだが、職種特化の数字は他にないので併用。

## 使う順
1. 経産省・IPA・総務省で土台を固める
2. NRI/PwC/INTAGE で生成AI側の最新数値
3. レバテック/doda で求人サイドの倍率と年収
4. 社内SEナビ／エージェント発の記事は最後にスキル要件・年収レンジ補強用
