社内SE16h／SIer業界SE21h／SES25-40hの3軸比較。年で最大528h(月44h差×12)、中央値ケース228h(月19h差×12)。

**Why:**
- パーソル2025年1月発表で社内SEが月16.0h(前回21.5h→大幅減)。3年連続全体平均も低下中。WLBアングルが今後しばらく刺さる
- SESには公的統計が存在しない。「平均9時間」(自社発信)〜「45時間」(業界推計)まで5倍の幅。代表値は二次ソースの三角測量で**月25h、繁忙期40-60h**を採用するのが安全
- 大手SIerの公表値(SCSK 3.3h・NEC 6.5h・日立 8.6h・富士通 9.1h)とOpenWork値(同じ会社が20-30h)の3-7倍乖離は**業界共通の落とし穴**。両方併記しないと信頼性を失う

**How to apply:**
- 残業比較記事では「3カラム+年取り戻し試算」が強い。最大ケース(528h/22日分/時給換算198万円)と中央値ケース(228h/9.5日分/85万円)を併記して幅を見せる
- 時給換算の前提は明示: 月20日勤務・所定160h・年収600万→時給3,750円
- SES数値は「25-40hレンジ・繁忙期60h」と幅で提示。単一値で断定しない
- 大手SIer4社の数字は引用OKだが、必ず「ただしOpenWork値は同社で20-30h」をセットで添える

**主要数値リファレンス:**
- 社内SE: パーソル2025年1月で**16.0h**(前回21.5h)／レバテック求人ベース17.8h(2024年12月)
- SIer業界SE: レバテック2024年12月で**21h**／SES総合研究所引用で**月27h**
- 大手SIer公表(2025年8月日経XTECH): SCSK 3.3h／NEC 6.5h／日立 8.6h／富士通 9.1h／NTTコムウェア 21.4h／BIPROGY 21.6h／SCSK公表別 22h
- SES: 公的統計なし／エン・ジャパン引用**約25h**(一次URL未確認)／繁忙期40-60h
- パーソル全体平均: 月20.6h(2025年4-6月実績)
- 残業多い職種1位: インフラコンサルタント／少ない職種1位: 医療事務(月10.5h)
- 有給取得率: NTTコムウェア95.1%／SCSK 88.0-89.4%／NTTデータグループ85.2%／全産業65.3%
- フルリモート: Job総研2025年1月で8.7%／IT エンジニア65%超(エンジニアナビ)

**検証で詰まる定番ポイント:**
- パーソル原本PDF(persol-career.co.jp/system/files/...)は403でWebFetch不可。プレスリリースHTML経由か二次媒体引用で押さえる
- SES代表値はエン・ジャパン引用が出回るが一次URLに辿れない。要追跡項目
- 大手SIerの「公表値vs現場値」乖離は必ず両方提示。片方だけだと信頼性低下
