社内SE求人票によく出る抽象的な誇大表現を、構造データに翻訳するための即引き辞書。1枚図解・記事・面接対策のいずれにも転用可能。

## 7大地雷ワードと翻訳

| 地雷ワード | 翻訳された実態 | 確認すべき逆質問 |
| --- | --- | --- |
| 幅広い業務 | ヘルプ+PCキッティング+企画+ベンダー対応を1人で | 直近半年の工数を5領域で内訳開示してほしい |
| 少数精鋭 | 1-3名のチーム、欠員未補充の可能性 | 直近2年の入退社数と正社員/業務委託/派遣の構成比 |
| アットホーム | 過干渉/業務外イベント/有給取りにくい遠回し | 情シス部門の有給取得率と平均残業時間 |
| 裁量権が大きい/自走できる | 教育もマニュアルも先輩もいない | オンボーディング設計とレポートライン |
| マルチタスク/変化を楽しめる | 同時並行5+ 割り込み多発 庶務+PJ兼務 | 進行中PJ数とPM/PL/担当者の分担 |
| DX推進・内製化を加速（具体なし） | 掛け声DX。運用保守100%のリスク | 過去3年のIT予算推移、内製/外製比率、内製チーム人数 |
| 経営に近い立場（レポートライン記載なし） | 総務部内の一機能。経営から遠い | 情シスのレポートラインとIT投資稟議の最終承認者、CIO/CDO職の有無 |

## これだけは見ろ5チェックリスト
1. 情シス部門の人員数と内訳（正社員/業務委託/派遣）
2. レポートライン（CIO直下／管理本部内／総務兼務）
3. IT予算規模 or 売上比率（製造業3.2%、金融6.5%、小売1.8%が参考）
4. 内製/外製の比率と主要委託先の有無
5. 時間配分内訳（運用保守/ヘルプ/企画/PJ）

## 主要数値（裏付け）
- 中堅中小24.5%がひとり情シス（ノークリサーチ2025/5・800社）
- 100-299名規模の20.2%が情シス1名以下（ソフトクリエイト2025）
- 100名未満の約4割がひとり情シス（複数調査の集計値）
- 公取委調査で自治体の98.9%が既存ベンダー再契約（ロックイン代理指標）
- 時間配分: 100-299名で問い合わせ対応28.8%、300-499名で37.7%

## How to apply
- 求人票×構造データの5軸点数化（明記2/推察可1/不明0）で7点以下は要注意
- 単独キーワードで断罪しない。「ワード×数字記述の有無」のセットで判定
- 景表法/名誉毀損リスクを避け、特定企業名で「ダメ」と書かない
- 体験談ベースの帰納的キーワードのため、実求人票の出現頻度実測は今後の課題
