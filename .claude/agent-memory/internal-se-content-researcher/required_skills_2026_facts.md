## 求人票頻出スキル(レバテック社内SE求人ページから抽出・2026年5月)
- クラウド: AWS(Lambda/RDS/CloudWatch/CloudFront), Azure, GCP, Salesforce
- セキュリティ: ゼロトラスト, ISMS, SSO/SAML/OIDC, Jamf, Intune, EDR
- AI/データ: 生成AI/LLM活用, データ分析基盤, Python, Go
- インフラ: ネットワーク設計, Win/Linux/macOS, Cisco Meraki
- 業務SaaS: Salesforce(Apex/Lightning), Workday, Slack, kintone
- 自動化: Terraform, Ansible, GitHub/GitOps, RPA

## 規模別必須スキルの傾向(2026年5月時点求人サンプル)
- 大手(1,000人〜): 専門分業, SAP S/4HANA(2027年問題), Salesforce/Workday上流, グローバルガバナンス
- 中堅(300-1,000人): クラウド+M365+セキュリティ+AI企画のT字型, Sky株式会社「AWS or Azure 1年以上」が標準ライン
- 中小(300人未満): ひとり情シス, M365中心, SaaSライセンス管理, ベンダー管理力

## 学習優先順位TOP5(コスパ順・複数ソース合意)
1. AWS SAA or Azure AZ-104(クラウド土台)
2. M365 + Entra ID + Intune(SaaS統合とゼロトラスト入口)
3. Copilot/RAG実装PoC経験
4. Power Platform(市民開発)
5. 情報処理安全確保支援士 or PMP(管理職トラック)

## 「便利屋化」防御の質問テンプレ(ITmedia久松剛 2025/12 ベース)
- 「DX人材の評価制度・KPIはあるか?」

- DX人材「大幅/やや不足」85.1%(IPA 日本) vs 米23.8%/独44.6%
- セキュリティ求人倍率42.6倍(レバテック2025/12)
