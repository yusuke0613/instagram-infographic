社内SE中途面接の頻出5問は固定化されており、SIer面接と評価軸が真逆。回答テンプレ系記事は「課題提起→経験＋数値→その会社固有→短期/中期貢献」の4要素テンプレに収束する。

**Why:** 2025-2026年は社内SE志望者の増加（doda入社決定37%最多）で1ポスト100-200倍超が常態化し、平凡回答では一次落ち連発。読者は「翌日の面接で使える具体回答例」を求めている（SIer/SES出身者が「技術アピールで撃沈」する実体験記事が複数）。

**How to apply:** 社内SE面接系の記事を書く時は以下を即引き：

- **頻出5問は固定**: ①なぜSIer/SESから社内SE ②なぜ当社か ③強み弱み ④ベンダーコントロール経験 ⑤入社後やりたいこと（出典：doda、社内SEナビ、CareerLift、ぼのぼの note、えんじこーむ note）
- **評価軸はSIerと真逆**: 技術＞人柄ではなく人柄＞技術／新技術＞既存ではなく既存改善＞新技術／スピード＞正確ではなく正確＞スピード（出典：ぼのぼの note 実体験）
- **NG回答の三大典型**: 「楽そう」「上流に行きたい」「Java5年AWSあります」（技術スタック羅列のみ）
- **採用担当が見ているもの**: 継続性（長く勤めるか）／再現性（過去実績を再現できるか）／カルチャーフィット。技術スタックで極端に評価しない（出典：zenn hassaku63 採用担当視点／R35 採用担当8年）
- **回答テンプレ4要素**: ①課題提起（IR/中期経営計画/採用ページから）②自分の経験＋数値（人数・予算・%・時間）③その会社固有の理由 ④短期(3-6ヶ月)→中期(1年)の貢献ステップ
- **逆質問は2-3個に絞る**: 「特にありません」は即減点。事業課題に踏み込む質問を1つは必ず入れる
- **業界別の重視軸**: メーカー=現場連携・SCM/ERP／金融=責任感・障害耐性／IT事業会社=内製・技術選定／医療人材=業務特化システム理解
