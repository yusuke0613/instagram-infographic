# 社内SE求人票の地雷ワード辞書 2026年版 — 「幅広い」を翻訳できれば入社後ギャップは8割消える

## 結論（一言で）
求人票のキレイな日本語を「人員・予算・体制」の数字に翻訳できれば、ハズレ社内SE求人は事前に消せる。

## なぜ今これを知るべきか（why_now）
2025年5月のノークリサーチ調査では中堅・中小企業の24.5%が「ひとり情シス」（前回比+3.2pt）。100名未満では約4割、100-299名でも20.2%が情シス1名以下と判明している。さらにITmedia 2025/12/10公開「IPA『DX動向2025』の裏側」は、DX人材が「便利屋化」「お手並み拝見」で潰れる構造を指摘。求人票の見出し美化と現場乖離はむしろ拡大している。

## 図解構成（推奨レイアウト）
- レイアウト案: **3列辞書テーブル**（左: 求人票の地雷ワード／中: 翻訳した本当の意味／右: 面接で確認する逆質問）+ 上部に「結論」+ 下部に「これだけは見ろチェックリスト5項目」
- 視線誘導: 上から下。各行は赤(地雷)→黄(翻訳)→緑(逆質問)で危険度の遷移を視覚化
- カラー配色案: 危険ゾーン=赤、デコード=黄、武器化=緑。背景は白で読みやすさ優先

## ブロック設計（3列辞書テーブル — 7行 + 補助ブロック）

### Row 1: 「幅広い業務をお任せ」
- ポイント: ジョブディスクリプション設計を放棄したサイン
- 翻訳: ヘルプデスク+PCキッティング+アカウント発行+ベンダー対応+企画まで全部担当。属人化前提
- 逆質問: 「直近半年の工数配分を5領域くらいで内訳お聞きしたいです（運用/ヘルプ/企画/プロジェクト/その他）」
- 出典: r35-se.com 求人票チェック / レバテック後悔6理由 / doda失敗事例

### Row 2: 「少数精鋭」
- ポイント: 人手不足の婉曲表現としてSIer/情シスで多用
- 翻訳: 1-3名のチームで欠員も補充できていない可能性。日刊SPAは「人が少なすぎて現社員がヒーヒー言っている」状態と解説
- 逆質問: 「直近2年の入退社人数と現在の正社員/業務委託/派遣の構成比を教えてください」
- 出典: 日刊SPA / ソフトクリエイト2025（ひとり情シス24.5%）

### Row 3: 「アットホームな職場」
- ポイント: 2000年代初頭から定番の警戒ワード
- 翻訳: 過干渉、業務外イベント強要、有給取りにくい、人事制度未整備の遠回し表現の場合あり
- 逆質問: 「直近3年の有給取得率と平均残業時間（情シス部門の数字）を教えていただけますか」
- 出典: 賢者の人事 ブラック企業キーワード編

### Row 4: 「裁量権が大きい」「自走できる人歓迎」
- ポイント: 「教える人もマニュアルもない」の婉曲
- 翻訳: 教育体制ゼロ・前任者退職済・属人化を新人で補修するパターン
- 逆質問: 「オンボーディングの設計と、相談先となる先輩・上長のレポートラインを具体的に教えてください」
- 出典: 賢者の人事 / r35-se.com 求人票チェック

### Row 5: 「マルチタスク」「変化を楽しめる方」
- ポイント: 役割分担崩壊・優先順位放棄のサイン
- 翻訳: 同時並行で5案件以上、割り込み多発、「プロジェクト推進」と「庶務」を1人で持つ
- 逆質問: 「現在進行中のプロジェクト数とPM/PL/担当者の分担を教えてください」
- 出典: ITmedia DX動向2025裏側「便利屋化」記述 / ソフトクリエイト時間配分(問い合わせ対応37.7%)

### Row 6: 「DX推進・内製化を加速」（具体記述なし）
- ポイント: 流行語の濫用。実態が運用保守100%のケースあり
- 翻訳: 内製化の進捗・予算・経営コミットが書かれていなければ「掛け声DX」。経営の本気度ゼロ
- 逆質問: 「過去3年のIT予算推移と、内製/外製の比率、内製チームの人数を教えてください」
- 出典: IPA DX動向2025（85.1%が人材不足）/ITmedia 2025/12/10 / 公取委 自治体98.9%が既存ベンダー再契約

### Row 7: 「経営に近い立場で活躍」（レポートライン記載なし）
- ポイント: 「総務部長の下→部長は非IT」など実質的に経営から遠いことが多い
- 翻訳: 情シスが管理本部の一機能として埋もれている。CIO/CDO直下でないとIT投資の意思決定に届かない
- 逆質問: 「情シスのレポートラインと、IT投資稟議の最終承認者は誰ですか。CIO/CDO職は設置されていますか」
- 出典: ITmedia DX動向2025裏側「お手並み拝見現象」/ r35-se.com

### 補助ブロック A: 「これだけは見ろ」5チェックリスト
求人票の以下5項目が「明記されている」会社を選べ。逆に1つでも欠けたら警戒度を1段上げる。
1. **情シス部門の人員数と内訳**（正社員/業務委託/派遣）
2. **レポートライン**（CIO直下／管理本部内／総務兼務）
3. **IT予算規模 or 売上比率**（製造業ベンチマーク3.2%、金融6.5%が参考値）
4. **内製/外製の比率と主要委託先の有無**
5. **時間配分の内訳**（運用保守/ヘルプデスク/企画/プロジェクト の概算%）

### 補助ブロック B: 求人票が"良いシグナル"を出している例（ポジティブ反転）
- 「IT予算 売上比3.5% / 部員12名（正社員8/業務委託4）/ CIO直下」
- 「PMO比率40%、ヘルプデスクは外部委託、内製アジャイルチーム5名」
- 「Microsoft 365 / ServiceNow / Tableau 利用、SaaS統廃合フェーズ」

## キーメッセージ（図解の周辺テキスト用）
- 「幅広い」は人員不足の翻訳語
- 100名未満の約4割がひとり情シス
- 求人票で数字が無い項目は面接で必ず聞く
- レポートラインが無い求人は経営直結の幻想
- 「アットホーム」より「予算と人員」を見ろ

## 落とし穴・注意点
- **キーワード単独で断罪しない**: 「アットホーム」が出てきても、人員数や予算が明示されていれば実態は健全な可能性。ワード×構造データ（人員/予算/レポートライン）の組み合わせで判断する
- **大企業ほど抽象化されやすい**: 大手の求人票は守備範囲が広く書かれがち。「幅広い」が必ずしも地雷とは限らない。組織図・配属部署を必ず確認
- **逆質問は「対立」でなく「興味」のトーン**: 数字を聞く時は「自分が活躍するイメージを持ちたく」など前置きを添える。詰問調はNG
- **業界規制**: 景表法・名誉毀損リスク回避のため、特定企業名で「この求人はダメ」とは言わない。「ワードの構造的傾向」として整理

## 次のアクション（CTA候補）
- 求人票を「人員/予算/レポートライン/内製比率/時間配分」の5項目で点数化（明記=2点、推察可=1点、不明=0点）
- 7点以下なら一次面接で必ず逆質問7本（このリストから）を投げる
- OpenWork・転職会議で情シス部門の口コミを最低5件読み比べ
- エージェントに「過去3年の入退社人数」「ひとり情シスかどうか」を直接質問

## 研究メモ（図解には載せないが diagrammer/reviewer が参考にする情報）

### データ確度メモ
- **ひとり情シス比率**: 出典で揺れあり。
  - ノークリサーチ2025/5: 中堅中小（年商500億円未満・800社）の24.5%（前回比+3.2pt）
  - ソフトクリエイト調査: 100-299名で20.2%、300-499名で3.3%が「1名以下」
  - 別調査で「100名未満は約4割」とされる（複数二次情報）
  - 図解では**「100名未満は約4割／中堅中小24.5%」**と幅で表記
- **公取委98.9%既存ベンダー再契約**は自治体ソフトウェア調達の数字。民間に直転用は注意だが、「ベンダーロックインの構造」を語る代理指標として使える
- **時間配分（問い合わせ対応37.7%）**はソフトクリエイト300-499名規模の数字。100-299名規模では28.8%。図解には30-37%と幅で表記推奨

### キーワード選定の根拠
求人票実調査ではなく、**転職メディア・ブラック企業見抜き記事・社内SE体験談記事の頻出ワード**から帰納的に抽出。実際のサンプル求人票での頻度測定までは未実施（次回課題）。

### 既存4本との差別化チェック
- ①SES 3年→社内SE 7つの現実: **キャリアパスの「現実」を語る** → 本案は「個別求人票の読み方」で完全別軸 ✅
- ②転職サービス3類型: **エージェント/サイトの選び方** → 本案は「求人票そのものの読み方」で完全別軸 ✅
- ③生成AI時代の社内SE求人動向: **市場マクロトレンド** → 本案はミクロな求人票読解 ✅
- ④大手SIer vs 大手メーカー社内SE: **企業類型比較** → 本案は会社問わずの読解術 ✅

### 矛盾するソース・未検証事項
