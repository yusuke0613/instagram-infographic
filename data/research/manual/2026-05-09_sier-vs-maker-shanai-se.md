# 大手SIer vs 大手メーカー社内SE — 比較表で見抜く「自分に向いているのはどっち？」

## 結論（一言で）
年収のピークはSIer、生活の安定はメーカー。出口戦略で選べ。

## なぜ今これを知るべきか（why_now）
2026年3月時点でITエンジニアの求人倍率は依然として高水準（dodaで2.39倍／レバテックでは11.6倍水準）で、社内SE志望はdoda登録者の希望職種で68%とトップ。「SIerからメーカー社内SE」は王道ルートだが、年収・スキル・出口で別物のキャリアになる。"今の選択"が10年後の市場価値を決める。

## 図解構成（推奨レイアウト）
- レイアウト案: **左右2カラム比較表**（左: 大手SIer / 右: 大手メーカー社内SE）。8行の比較行 + 上部に「結論」、下部に「自分はどっち？4つの問い」
- 視線誘導: 上から下へ。各行は左右で対比が成立する短文に揃え、最も差の出る「年収ピーク」「スキル」「評価」のセルを色強調
- カラー配色案: 左SIer=ブルー系（外向き・受託）、右メーカー=グリーン系（内向き・事業）

## ブロック設計（比較表の各行）

### Row 1: 30代モデル年収レンジ
- ポイント: 平均は近いが、ピークでSIerが伸びる
- 詳細:
  - **SIer**: NTTデータ平均906万円(平均39.9歳)／富士通929-965万円(平均43.6歳)／NRI1,156-1,322万円。30代は650-1,000万円帯、PM昇格でNTTデータPM平均971万円
  - **メーカー社内SE**: 日立製作所SE職平均736-786万円(レンジ360-1,700万)／トヨタシステムズ平均約750万円(平均31歳)。30代は550-850万円が中央値
- 出典: ワンキャリア転職、ASSIGNメディア、OpenWork、TalentSquare

### Row 2: 賞与・残業代の構造
- ポイント: SIerは残業代依存、メーカーは賞与で月収を維持
- 詳細:
  - **SIer**: 基本給は控えめで「賞与+残業代」で総額が決まる構造（NTTデータの社員クチコミ）
  - **メーカー社内SE**: 賞与年2回が大きく、業績連動。残業代は労使協定で抑制傾向
- 出典: OpenWork NTTデータ／日立製作所

### Row 3: 平均残業時間（月）
- ポイント: 社風次第だが、ホワイト大手はSIerの方が短いケースも
- 詳細:
  - **SIer**(大手): SCSK 3.3h／NEC 6.5h／日立 8.6h／富士通 9.1h(月10h未満上位)。中堅SIerは20-30h、繁忙期は60h+も
  - **メーカー社内SE**: 業界平均SE 21h／全産業平均20.6h(2025年パーソル調査)。「納期は緩いがトラブル時は跳ねる」
- 出典: 日経XTECH、パーソルキャリア2025、レバテック

### Row 4: リモート・有給取得率
- ポイント: 制度はメーカー優位、運用はSIer大手も健闘
- 詳細:
  - **SIer**: NTTコムウェア有給95.1%／SCSK 88.0%。客先常駐案件はリモート不可になるリスクあり
  - **メーカー社内SE**: 有給18.1日/年(全産業平均)取得率高め。本社勤務固定でリモート併用が標準
- 出典: 35歳からの社内SE転職、厚労省就労条件総合調査2025

### Row 5: 業務内容（時間配分の質）
- ポイント: SIer=作る／メーカー社内SE=決める・任せる
- 詳細:
  - **SIer**: 顧客ヒアリング→要件定義→設計→開発→テスト→運用。手を動かす比率が高くコードも書く
  - **メーカー社内SE**: 業務部門の要望整理→IT企画→ベンダー選定・コントロール→プロジェクト推進→運用統制。"PCトラブル何でも屋"側面も
- 出典: ドライブライン、クロスネットワーク、dodaエンジニア、レバテック後悔6理由

### Row 6: 身につくスキル/キャリア資産
- ポイント: SIer=技術ポータブル、メーカー=業務知識+調整力
- 詳細:
  - **SIer**: 多業界の最新技術／PM経験／プログラミング実務。市場で「即戦力」表記されやすい
  - **メーカー社内SE**: 自社業務ドメイン知識(製造/物流/会計など)／ベンダー管理／社内政治力。汎用化しにくい資産
- 出典: ドライブライン、ユニゾンキャリア、レバテック後悔6理由

### Row 7: 評価制度・昇進スピード
- ポイント: 成果のSIer、年功+業績のメーカー
- 詳細:
  - **SIer**: 実力主義寄り。NRIは28歳前後で1,000万円到達例。マネジメント昇格で急加速
  - **メーカー社内SE**: 給与テーブル依存で年功色強め。日立も「昇給は年功序列」のクチコミ。トヨタ系は昇格にTOEIC600以上などの要件
- 出典: ワンキャリア転職NRI、OpenWork日立、TalentSquareトヨタシステムズ

### Row 8: 転職市場での出口
- ポイント: SIerは出口広い、メーカー社内SEは"次"が悩ましい
- 詳細:
  - **SIer→**: 社内SE/コンサル/自社開発/フリーランス全方位。市場価値が落ちにくい
  - **メーカー社内SE→**: 同業他社の社内SEは行きやすいが、SIer/開発職への戻りはコード経験で見劣りしやすい
- 出典: タイズマガジン、レバテック後悔6理由、ドライブライン

## キーメッセージ（図解の周辺テキスト用）
- 平均年収は近い、ピーク年収はSIer
- 安定・WLB重視ならメーカー社内SE
- 「出口の広さ」で選ぶならまだSIerに残れ
- 業務知識を武器にするならメーカー一択

## 落とし穴・注意点
- **「社内SE=楽」は神話**: 一人情シス的な業務量、トラブル何でも屋、PC配布などの庶務も多い(レバテック後悔6理由 88%が中小では1人以下)
- **メーカー社内SE=高給ではない**: 大手電機の平均は700-900万円台。SIer上位(NRI/NTTデータ)の方が伸びる
- **企業差が業態差より大きい**: 同じ"メーカー社内SE"でもトヨタシステムズ(子会社型)と日立本体(事業会社内IT)は別物。配属部署で残業もリモート可否も変わる
- **数字は2024-2025年公開ベース**: 賃上げ局面で来年さらに上振れ可能性。年収は有報・OpenWorkの最新値を必ず確認

## 次のアクション（CTA候補）
- 自分の「10年後の市場価値」を3パターン書き出す → SIerに残る/メーカー社内SE/コンサル
- 志望メーカーの**情シス子会社か事業会社直下IT部門か**を採用ページで確認(働き方が大きく違う)
- OpenWorkで志望企業の「ワークライフバランス」「待遇満足度」スコアを必ず照合
- 自分はどっち？4つの問いに答えて適性を判定:
  1. 「コードを5年後も書いていたい」 → SIer
  2. 「同じ事業を10年深く支えたい」 → メーカー社内SE
  3. 「年収ピーク1,200万+を狙いたい」 → SIer(NRI/外資SIer)
  4. 「家族時間と健康診断のA判定が最優先」 → メーカー社内SE

## 研究メモ（図解には載せないが diagrammer/reviewer が参考にする情報）

### データ確度メモ
- **NTTデータ平均年収**: 有報ベース906万円(平均39.9歳, 2024年度) ／ OpenWork上は771万円(回答者バイアス)。図解では有報ベースを採用
- **富士通**: 929万円(2026年版ASSIGN) と 965万円(別調査) で揺れ。中央値で表記
- **トヨタシステムズ平均31歳で750万円**は若年集団のため、30代後半まで進めばさらに上振れ可能
- **日立製作所SE/開発職736-786万円**は職種別中央値。本体直雇いの社内IT部門の正確な数字は公開なし。研究中項目

### 業態の細分化メモ（図解では省略）
"メーカー社内SE"は実は3パターンに分かれる:
1. **本体直下のIT部門/DX推進部**(日立本体、ソニー、三菱電機など) — 年収高、ベンダーコントロール+企画
2. **情シス子会社**(トヨタシステムズ、パナソニックインフォメーションシステムズなど) — 年収やや控えめ、開発比率高め
3. **事業部内のSE**(製造ライン専属など) — ドメイン知識特化

これを図解に入れると密度オーバーするので、CTAの「子会社か直下か確認」だけ残した。

### 検証で詰まった点
- 大手メーカーの「社内SE職限定の30代年収」は公開データが乏しい。OpenWorkの職種別中央値で代用
- リモートワーク実施率の業界別細分(IT 17.0%+2.2pt が最新値だが、大手SIer/大手メーカーIT部門の細分はなし)
- 「社内SE求人倍率」単独の正確な数字は転職エージェント独自データで会員限定。dodaの「希望職種で社内SE 68%トップ」が代理指標

### 矛盾するソース
- 「平均残業時間」: 大手SIerの公表値(8-9h)とOpenWorkクチコミ(20-30h、繁忙期60h+)で大きな差。**部署差が極端に大きい**ことを落とし穴に明記
- 「年収」: 社内SE 685万 vs SIer 558万(ユニゾンキャリア集計)とSIer上位は1000万超の構図は両立する。「平均では社内SEが上、ピークではSIerが上」というメッセージで整合させた

- 8行は1枚図解には多すぎる可能性。優先順位は「年収/残業/業務内容/スキル/出口」の5行を必須、残りは余白次第で
- 各セルの文言は12-18字に圧縮することを推奨（並列構造を優先）
